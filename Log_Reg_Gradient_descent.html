Define Model¶

Logistic sigmoid function¶

Since we are dealing with a target that has a range from 0 to 1, we need to squash the outcomes into the interval $[0,1]$ by using the logistic sigmoid function, which is defined as

$$ \sigma(\mathbf{w},\mathbf{X}) = \frac{1}{1 +e^ {-\mathbf{w}\cdot \mathbf{X}}}$$

where $ \mathbf{w} $ is the weight vector and $ \mathbf{X} $ is the data.

In [3556]:
def sigmoid(w, X):
    return (1. / (1. + np.exp(-np.dot(X,w))))

Loss function¶

We use the cros entropy loss function for logistic regression and include the regularizer term as the following

$$ Loss = \frac{1}{m}\big[- \mathbf{t}^{T}\ln(\sigma(\mathbf{w},\mathbf{X})) - (\mathbf{1} - \mathbf{t})^{T} \ln(\mathbf{1}-\sigma(\mathbf{w},\mathbf{X})\big] + \frac{\lambda}{2m}\mathbf{w}^{T}\mathbf{w} $$
In [3557]:
def loss(w, X, t, lam):
    m = X.shape[0]
    y = sigmoid(w,X)  
    J = (1. / m) *(- np.dot(t.T,np.log(y)) - np.dot((1. - t.T),np.log(1. - y)))
    regularizer = (lam / (2. * m)) * np.dot(w.T, w)
    J += regularizer
    return J

The partial derivative of the loss function with respect to the vector $\mathbf{w}$ is

$$ \nabla_{\mathbf{w}} Loss = \frac{1}{m}[(\sigma(\mathbf{w},\mathbf{X}) - \mathbf{t})^{T}\mathbf{X}]^T + \frac{\lambda}{m}\mathbf{w} $$
In [3558]:
def partial(w, X, t, lam):
    m = X.shape[0]
    y = sigmoid(w, X)
    d = (1. / m) * np.dot((y - t).T, X).T  
    reg = (lam / m) * w
    d = d + reg
    return d

Learning rule¶

Our learning rule is \begin{align} \mathbf{w}{t+1} &= \mathbf{w}{t} - \eta \nabla\mathbf{w} Loss \ &= \mathbf{w}{t} - \eta0 t^{-\alpha} \nabla\mathbf{w} Loss \end{align}

where $\eta_0$ is a constant, $t$ is the iteration number, and $\alpha = .9$.

In [3559]:
def update_parameter(w, d, iteration, rate = .2, a = -.9):
    w = w - rate * (iteration ** a) * d
    return w

Gradient Descent¶

The gradient updates w until the loss converges.

In [3560]:
#define gradient descent
def gradient_descent(w, X, t, lam, max_step = 1000, convergence = 0.00001):
    counter = 1
    L = loss(w, X, t, lam)
    L_prev = L + 10
    #update w until loss converges
    while(abs(L - L_prev) > convergence) and (counter <= max_step):
        L_prev =  L
        L = loss(w, X, t, lam)
        d = partial(w, X, t, lam)
        w = update_parameter(w, d, counter)
        counter += 1
        
    w_norm = np.linalg.norm(w)
    results = {"Weights": w, "Limit" : L, "W_norm": w_norm}
    return results

    lam = start
      
    results = {}

        #w = np.random.random([max_features])

        for train_index, test_index in kf.split(X):
            X_train, V, X_target, V_target = X[train_index, :], X[test_index, :], t[train_index], t[test_index] 
            gd = gradient_descent(w, X_train, X_target, lam)
            w = gd["Weights"]
            w_n = gd["W_norm"]
            validation_loss  = loss(w, V, V_target, lam = 0)
            loss_results.append(validation_loss)
            w_results.append(w_n)
        #average the loss    
        L = np.mean(loss_results)
        W = np.mean(w_results)
        results[lam] = [L,W]
        lam += step_size
    return (results)
